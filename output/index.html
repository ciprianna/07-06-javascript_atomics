What's the difference between alert() and console.log()?

An alert is a pop-up window that the user must acknowledge before moving on. A console.log() message is never shown to the user. It stores message content into a log for the developer to view.

What's one situation where you might use an `alert()` to tell the user something?

If there was a warning message, such as leaving the page removes clears all unsaved data. An alert might also be used to tell the user that information is missing from a form when they try to submit information.

What's one way that `alert()` might get obnoxious?

If there are multiple alerts on a page, it can probably seem spammy. They should probably be limited to only situations where the user should really be paying attention to something vital.

Would it make sense to use console.log() to tell the user things? Why or why not?

It wouldn't really make sense to use the console.log() to tell the user information because it's not actively shown on the screen. Instead, it keeps track of the statements in a log that could be looked up if needed.

What's one way that the console.log() might be a good tool for you to use?

A log could be a good tool to keep track of analytics on a site. For example, it might track the clicks that a user takes (i.e., from the home page, a large percentage of users click on View X.). Helps track progression of users on a website.

When do the console.log() statements get run? Is it before the first alert gets displayed? After it gets displayed but before the user clicks "OK" (or equivalent)? After the user clicks to close the dialog box? A different time? How did you find out for sure?

The console.log() gets run after the user clicks to close the dialog box. I figured this out by inspecting the element in the web browser, and clicking on the console tab at the top. When the page is first loaded, the log is empty, even as the first dialog box appears. After I clicked "OK", the log filled and the second dialog box appeared on the screen. It runs in the order as written in the .js file.
